# CollabHub — 3D Image Generation Prompts

> **Brand:** CollabHub
> **Aesthetic:** Bright & Playful (Colorful 3D Shapes)
> **Palette:** Purple (#7C3AED) + Blue (#3B82F6) + Lavender (#C4B5FD) + Cyan (#67E8F9) + White
> **Style Keywords:** Glossy, rounded, soft shadows, Pixar-like, isometric, playful, clay-render

---

## Quality Tags (Append to Every Prompt)

```
3D render, octane render, bright studio lighting, soft shadows,
4K, ultra detailed, no text, no watermark
```

**Aspect Ratios:**
- 16:9 → Hero sections, backgrounds, banners
- 1:1 → Feature illustrations, icons, cards
- 9:16 → Mobile showcases, app store assets, Flutter splash

## Recommended Generation Tools

- **Midjourney v6** — Best for 3D illustration quality
- **Leonardo AI** — Good for consistent 3D style with fine-tuning
- **Dall-E 3** — Quick iterations, good prompt understanding
- **Stable Diffusion + 3D LoRA** — Free option with custom models
- **Spline 3D** — For interactive 3D elements (web only)
- **Rive / Lottie** — For animated assets (loading, transitions)
